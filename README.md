# Green Dial Real Marketplace v1

This version removes fake watches and fake users.

What changed:
- No fake nearby listings
- No fake seller profiles
- Empty states until real watches are added
- Add Watch form saves real user-added listings in browser localStorage
- Profile form saves real user profile in browser localStorage
- Buy Nearby, My Listings, Watch Detail, Messages, Saved, Authentication, Market Index pages are ready as real marketplace flows

Important:
This is still a static frontend prototype. For public launch with real users across devices, connect it to a backend such as Firebase or Supabase for accounts, database, image storage, messaging, and location search.
